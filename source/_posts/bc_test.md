## 使用:
Ctrl + K:用来调出生成代码框,输入你想要的程序

Ctrl + L:问它任何问题,比如代码怎么用,第几行代码是什么意思等等


Ctrl + Enter 接受代码

Ctrl + Backspace 不接受，可修改描述

Ctrl + N 修改描述

Ctrl + K 继续添加描述
